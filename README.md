# 💰 Smart Budget Allocator & Bill Splitter

A beginner-friendly web application that helps users:
- Allocate monthly income into essentials, lifestyle & savings
- Split bills equally among friends
- Store past bill split history using localStorage
- Visualize budget using a pie chart

## 🚀 Features
- Budget allocation using 50-30-20 rule
- Interactive pie chart (Chart.js)
- Bill splitting for group expenses
- Event history stored in browser
- Responsive & clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
